# LeetCode-Cpp
开发中...

## 文件夹介绍
1. `/.vscode`: vscode配置文件
2. `/src`: leetcode库
3. `/answer`: 习题解答
4. `/tests`: 测试文件
5. `/bin`: 二进制文件


## Run
1. run 0001.cpp
    ```bash
    # 环境: ubuntu
    mkdir ./bin
    /usr/bin/g++-11 -fdiagnostics-color=always -g ./answer/0001.cpp  -o ./bin/0001 -I ./src
    ./bin/0001
    ```

2. use leetcode库
    ```cpp
    #include <leetcode>
    ```
